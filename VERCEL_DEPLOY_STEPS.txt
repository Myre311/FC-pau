ÉTAPE 5 : DEPLOY !
──────────────────────────────────────────────────────────────
Cliquer sur le gros bouton bleu "Deploy"

⏱️ Durée : ~3-5 minutes

Vercel va :
1. Cloner le repo GitHub
2. Installer les dépendances
3. Générer Prisma Client
4. Builder Next.js avec toutes les pages
5. Déployer sur CDN global

ÉTAPE 7 : CORRIGER L'URL (si nécessaire)
──────────────────────────────────────────────────────────────
Si tu as mis une URL temporaire :

1. Dashboard Vercel → Ton projet → Settings → Environment Variables
2. Éditer NEXT_PUBLIC_SITE_URL → mettre la vraie URL Vercel
3. Éditer ALLOWED_ORIGINS → mettre la vraie URL Vercel
4. Deployments → Redeploy (latest)
